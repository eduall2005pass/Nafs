## ✨ Features

### 🔒 Security Core
- **5-Layer Uninstall Prevention**
  - Instant detection (50ms response)
  - Multiple BACK actions
  - Auto-cancel button clicking
  - All OEM support (Samsung, Xiaomi, Huawei, etc.)
  - 8-second persistent overlay

- **Continuous Settings Monitoring**
  - 300ms scan interval
  - NafsShield settings protection
  - Force Stop prevention
  - Disable prevention
  - Clear Data prevention

### 🔐 PIN Protection System
- Keyword deletion requires PIN
- Keyword disable requires PIN
- App unblock requires PIN
- Browser removal requires PIN
- Accidental action prevention

### 📱 App Management
- Block apps with visual icons
- Search functionality
- Package name verification
- Real-time filtering
- Smooth RecyclerView

### 🌐 Browser & WebView Control
- Custom browser whitelist
- In-app WebView detection
- Any app as browser support
- Automatic WebView blocking

### ⏰ Schedule System
- Custom time-based blocking
- Start/end time configuration
- Day of week selection
- Multiple schedules support
- Toggle on/off
